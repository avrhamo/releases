# Installation Instructions

## Windows

1.  Locate the installer file: `dist-electron/API Workspace Setup <version>.exe`.
2.  Double-click the `.exe` file to run the installer.
3.  Follow the on-screen instructions.
4.  The application will be installed and a shortcut will be created on your desktop.

## macOS

1.  Locate the disk image file: `dist-electron/API Workspace-<version>.dmg`.
2.  Double-click the `.dmg` file to open it.
3.  Drag the "API Workspace" icon to the "Applications" folder.
4.  You can now launch the application from your Applications folder.

## Linux

1.  Locate the AppImage file: `dist-electron/API Workspace-<version>.AppImage`.
2.  Make the file executable: `chmod +x API Workspace-<version>.AppImage`.
3.  Run the file: `./API Workspace-<version>.AppImage`.

## Troubleshooting

If you encounter any issues during installation, please ensure you have the necessary permissions to install applications on your system.
